# 激活步骤：

	2.	弹出登录界面，点击『Continue without account』蓝色文字（没有此步的话请跳过）
	3.	点击任务栏的(桌面右上角)「一个圆圈和一把锁的图标」，再点击「设置」，再点击「关于」，然后点击「注册」，输入以下激活码，点击提交即可。
